## 项目概览

这是一个基于 **Next.js + Rust** 的全栈博客平台，支持：

- 📝 BlockNote 富文本编辑器
- 🔍 全文搜索
- 🎨 动态主题系统
- 📊 化学分子式渲染
- 🔐 基于 JWT 的认证系统
- 🚀 Docker Compose 一键部署
