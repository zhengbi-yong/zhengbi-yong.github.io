## 概述（Overview）

Weed detection holds significant research importance and practical value, particularly in the realm of agriculture where it can greatly enhance crop yield and quality. The primary goal of weed detection is to accurately identify and differentiate weeds from crops, which is crucial for targeted weed management strategies that minimize the use of herbicides and maximize the efficiency of manual or mechanical weed control.

杂草检测具有重大的研究重要性和实用价值,特别是在农业领域,它可以大大提高作物产量和质量。杂草检测的主要目标是准确识别杂草并将其与作物区分开来,这对于最大限度地减少除草剂的使用并最大限度地提高手动或机械杂草控制效率的有针对性的杂草管理策略至关重要。

Through this project, we aim to learn and apply cutting-edge technologies in computer vision to achieve accurate weed detection.

通过该项目,我们的目标是学习和应用计算机视觉中的尖端技术,以实现准确的杂草检测。

## 评价方式（Evaluation）

This competition is evaluated on the precision at intersection over union (IoU) thresholds set as 0.5.

该竞赛根据设置为 0.5 的并集（IoU）阈值交叉点的精度进行评估。

The precision value is calculated based on the number of true positives (TP), false negatives (FN), and false positives (FP) resulting from comparing the predicted object to all ground truth objects:

精度值通过真阳性（TP）、假阴性（FN）和假阳性（FP）的数量进行计算, 这些数量是由预测对象与所有真值对象进行比较而产生的:

$$
precision = \frac{TP}{TP+FP+FN}.
$$

A true positive is counted when a single predicted object matches a ground truth object with an IoU above the threshold. A false positive indicates a predicted object had no associated ground truth object. A false negative indicates a ground truth object had no associated predicted object.

当单个预测对象与 IoU 高于阈值的真值对象匹配时,计算真正数。假阳性表示预测对象没有关联的地面真值对象。假阴性表示地面真值对象没有关联的预测对象。

为了能够更好地兼容目前主流的目标检测模型，例如YOLO，我们需要将标签转换为YOLO格式。我们首先需要将圆形的标注转换为矩形的标注，然后我们需要生成对应的txt格式的YOLO标签。
一个将上述类型标签转换为YOLO格式标签的程序如下：

```python
import os
import json

def circle_to_bbox(shape, img_width, img_height, class_id):
    # 提取圆心和半径
    x_center, y_center = shape['points'][0]
    x_radius, y_radius = shape['points'][1]
    radius = ((x_radius - x_center) ** 2 + (y_radius - y_center) ** 2) ** 0.5

    # 计算矩形框的左上角和右下角
    x_min = max(0, x_center - radius)
    y_min = max(0, y_center - radius)
    x_max = min(img_width, x_center + radius)
    y_max = min(img_height, y_center + radius)

    # 转换为YOLO格式：相对坐标
    x_center_normalized = x_center / img_width
    y_center_normalized = y_center / img_height
    width_normalized = (x_max - x_min) / img_width
    height_normalized = (y_max - y_min) / img_height

    return f"{class_id} {x_center_normalized} {y_center_normalized} {width_normalized} {height_normalized}"

def convert_annotations_to_yolo(input_dir, output_dir):
    # 如果输出目录不存在，创建它
    if not os.path.exists(output_dir):
        os.makedirs(output_dir)

    # 遍历输入文件夹中的所有json文件
    for file_name in os.listdir(input_dir):
        if file_name.endswith('.json'):
            input_file_path = os.path.join(input_dir, file_name)
            output_file_path = os.path.join(output_dir, file_name.replace('.json', '.txt'))

            # 打开并解析json文件
            with open(input_file_path, 'r') as json_file:
                data = json.load(json_file)

            img_width, img_height = data['imageWidth'], data['imageHeight']
            shapes = data['shapes']

            yolo_labels = []
            for shape in shapes:
                # 根据标签的内容设置 class_id
                if shape['label'] == 'mq':
                    class_id = 1
                else:
                    class_id = 0

                if shape['shape_type'] == 'circle':
                    yolo_label = circle_to_bbox(shape, img_width, img_height, class_id)
                    yolo_labels.append(yolo_label)

            # 将YOLO格式的标签写入到新的txt文件中
            with open(output_file_path, 'w') as yolo_file:
                for label in yolo_labels:
                    yolo_file.write(label + '\n')

    print(f"转换完成！所有标签已保存到 {output_dir}")

# 示例用法：
input_folder = "D:\\WeedDetection\\dataset\\train\\train\\labels"  # 输入文件夹路径，包含json格式的标签
output_folder = "D:\\WeedDetection\\dataset\\train\\train\\labels_yolo"  # 输出文件夹路径，保存转换后的YOLO标签
# input_folder = "D:\\WeedDetection\\dataset\\test\\test\\labels"  # 输入文件夹路径，包含json格式的标签
# output_folder = "D:\\WeedDetection\\dataset\\test\\test\\labels_yolo"  # 输出文件夹路径，保存转换后的YOLO标签
convert_annotations_to_yolo(input_folder, output_folder)

```
