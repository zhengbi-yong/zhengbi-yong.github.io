## 基本工作流程

我们在使用该框架进行深度学习项目开发时，只需要按照下述步骤进行定制即可：

1. 编写 PyTorch Lightning module

3. 编写实验配置，包含模型和 datamodule 的路径

## 1. 编写 PyTorch Lightning module

官方文档：[LightningModule](https://lightning.ai/docs/pytorch/stable/common/lightning_module.html)

LightningModule 将 PyTorch 代码组织成 6 个部分:

1. 初始化 (`__init__` 和 `setup()`)
2. 训练循环 (`training_step()`)
3. 验证循环 (`validation_step()`)
4. 测试循环 (`test_step()`)
5. 预测循环 (`predict_step()`)
6. 优化器和学习率调度器 (`configure_optimizers()`)

| 名称                     | 描述                                                    |
| ------------------------ | ------------------------------------------------------- |
| `__init__` 和 `setup()`  | 在此初始化                                              |
| `forward()`              | 模型对数据进行前向传播的计算（与 `training_step` 分开） |
| `training_step()`        | 完整的训练步骤                                          |
| `validation_step()`      | 完整的验证步骤                                          |
| `test_step()`            | 完整的测试步骤                                          |
| `predict_step()`         | 完整的预测步骤                                          |
| `configure_optimizers()` | 定义优化器和学习率调度器                                |
