## 引言

Vim 是一款功能强大的文本编辑器，以其高效的键盘操作和独特的模态编辑方式而闻名。掌握 Vim 的快捷键不仅可以显著提升文本编辑效率，更能让您体验到"编辑即思考"的流畅工作流。与传统的编辑器不同，Vim 通过组合简单的命令来实现复杂的编辑操作，这种设计哲学使得熟练的 Vim 用户可以仅用键盘就能完成几乎所有编辑任务，无需频繁使用鼠标。

Vim 的学习曲线虽然相对陡峭，但一旦掌握了其核心概念和常用快捷键，您会发现编辑效率的提升是显著的。许多开发者将 Vim 视为提高生产力的重要工具，甚至有人表示"一旦习惯了 Vim，就很难再回到传统的编辑器"。

本文档旨在提供一份全面而实用的 Vim 快捷键参考指南，从基础的光标移动到高级的宏录制，从简单的文本编辑到复杂的代码重构，帮助您系统性地学习和掌握 Vim 的强大功能。无论您是刚开始接触 Vim 的新手，还是希望深入理解 Vim 高级特性的资深用户，都能从本文中找到有价值的内容。

**学习建议**

- **循序渐进**：先掌握基础的光标移动和文本编辑，再逐步学习高级功能
- **实践为主**：快捷键的记忆需要通过实际使用来巩固，建议在日常编辑中刻意练习
- **理解原理**：理解 Vim 的操作符-动作组合模式，可以事半功倍
- **建立习惯**：将 Vim 快捷键融入日常工作流，形成肌肉记忆

## 第一部分：Vim 模式基础

Vim 的核心特性之一是其多模式编辑系统。理解不同的模式及其切换方式是高效使用 Vim 的基础。

### 主要模式

**普通模式（Normal Mode）**

普通模式是 Vim 启动后的默认模式，也是 Vim 的核心模式。在这个模式下，键盘上的每个键都对应一个命令，用于移动光标、删除文本、复制粘贴等操作。普通模式的设计理念是让用户能够快速执行编辑操作，而无需使用鼠标或复杂的组合键。

**插入模式（Insert Mode）**

插入模式用于输入文本，类似于传统编辑器的编辑状态。在这个模式下，您可以直接输入字符，键盘上的字母和数字会直接插入到文档中。插入模式适合进行文本输入和修改。

**可视模式（Visual Mode）**

可视模式用于选择文本范围，类似于传统编辑器中的文本选择。Vim 提供了三种可视模式：字符可视模式、行可视模式和块可视模式，分别用于选择不同范围的文本。

**命令行模式（Command-Line Mode）**

命令行模式用于执行 Vim 命令，如保存文件、退出编辑器、搜索替换等。通过命令行模式，您可以执行复杂的操作和配置。

### 模式切换

| 快捷键   | 功能                   | 说明                               |
| -------- | ---------------------- | ---------------------------------- |
| `i`      | 进入插入模式（光标前） | 在光标位置前插入文本               |
| `a`      | 进入插入模式（光标后） | 在光标位置后追加文本               |
| `I`      | 进入插入模式（行首）   | 在行首第一个非空字符前插入         |
| `A`      | 进入插入模式（行尾）   | 在行尾追加文本                     |
| `o`      | 在下方新行插入         | 在当前行下方新建一行并进入插入模式 |
| `O`      | 在上方新行插入         | 在当前行上方新建一行并进入插入模式 |
| `v`      | 进入字符可视模式       | 按字符选择文本                     |
| `V`      | 进入行可视模式         | 按行选择文本                       |
| `Ctrl-v` | 进入块可视模式         | 按块选择文本（列选择）             |
| `:`      | 进入命令行模式         | 执行 Vim 命令                      |
| `Esc`    | 返回普通模式           | 从任何模式返回普通模式             |
| `Ctrl-[` | 返回普通模式           | Esc 的替代键                       |

**模式切换技巧**

- 在插入模式下，按 `Esc` 或 `Ctrl-[` 可以快速返回普通模式
- 使用 `Ctrl-o` 可以在插入模式下临时执行一个普通模式命令，然后自动返回插入模式
- 在可视模式下选择文本后，可以使用操作符（如 `d`、`y`、`c`）对选中的文本进行操作

## 第二部分：光标移动

高效的光标移动是 Vim 编辑效率的基础。Vim 提供了多层次的光标移动方式，从字符级到文件级，满足各种编辑需求。

### 字符级移动

| 快捷键 | 功能         | 说明         |
| ------ | ------------ | ------------ |
| `h`    | 左移一个字符 | 向左移动光标 |
| `l`    | 右移一个字符 | 向右移动光标 |
| `j`    | 下移一行     | 向下移动光标 |
| `k`    | 上移一行     | 向上移动光标 |

**记忆技巧**：`h`、`j`、`k`、`l` 对应键盘上的左手位置，`j` 向下（像字母 j 的下半部分），`k` 向上。

### 单词级移动

| 快捷键 | 功能                                   | 说明                             |
| ------ | -------------------------------------- | -------------------------------- |
| `w`    | 移动到下一个单词开头                   | 跳转到下一个单词的开始位置       |
| `W`    | 移动到下一个单词开头（空格分隔）       | 以空格为分隔符，跳转到下一个单词 |
| `e`    | 移动到当前或下一个单词结尾             | 跳转到单词的结束位置             |
| `E`    | 移动到当前或下一个单词结尾（空格分隔） | 以空格为分隔符的单词结尾         |
| `b`    | 移动到当前或上一个单词开头             | 向后跳转到单词开始位置           |
| `B`    | 移动到当前或上一个单词开头（空格分隔） | 以空格为分隔符的单词开头         |
| `ge`   | 移动到上一个单词结尾                   | 向后跳转到单词结束位置           |

**单词 vs 空格分隔单词**：小写字母（`w`、`e`、`b`）以标点符号和空格为分隔符，大写字母（`W`、`E`、`B`）仅以空格为分隔符。

### 行内移动

| 快捷键    | 功能                       | 说明                               |
| --------- | -------------------------- | ---------------------------------- |
| `0`       | 移动到行首                 | 跳转到行的第一个字符               |
| `^`       | 移动到行首第一个非空字符   | 跳转到行的第一个非空白字符         |
| `$`       | 移动到行尾                 | 跳转到行的最后一个字符             |
| `g_`      | 移动到行尾最后一个非空字符 | 跳转到行的最后一个非空白字符       |
| `f{char}` | 向前查找字符               | 在当前行向前查找指定字符并跳转     |
| `F{char}` | 向后查找字符               | 在当前行向后查找指定字符并跳转     |
| `t{char}` | 向前查找字符（之前）       | 跳转到指定字符之前的位置           |
| `T{char}` | 向后查找字符（之后）       | 跳转到指定字符之后的位置           |
| `;`       | 重复上次查找               | 重复上一次 `f`、`F`、`t`、`T` 操作 |
| `,`       | 反向重复上次查找           | 反向重复上一次查找操作             |

### 屏幕滚动

| 快捷键   | 功能           | 说明                   |
| -------- | -------------- | ---------------------- |
| `Ctrl-f` | 向下滚动一屏   | 向前滚动一屏内容       |
| `Ctrl-b` | 向上滚动一屏   | 向后滚动一屏内容       |
| `Ctrl-d` | 向下滚动半屏   | 向前滚动半屏内容       |
| `Ctrl-u` | 向上滚动半屏   | 向后滚动半屏内容       |
| `Ctrl-e` | 向下滚动一行   | 屏幕向下滚动一行       |
| `Ctrl-y` | 向上滚动一行   | 屏幕向上滚动一行       |
| `zz`     | 当前行居中     | 将当前行移动到屏幕中央 |
| `zt`     | 当前行置顶     | 将当前行移动到屏幕顶部 |
| `zb`     | 当前行置底     | 将当前行移动到屏幕底部 |
| `H`      | 移动到屏幕顶部 | 跳转到屏幕的第一行     |
| `M`      | 移动到屏幕中间 | 跳转到屏幕的中间行     |
| `L`      | 移动到屏幕底部 | 跳转到屏幕的最后一行   |

### 文件级跳转

| 快捷键 | 功能             | 说明                                 |
| ------ | ---------------- | ------------------------------------ |
| `gg`   | 跳转到文件开头   | 跳转到文件的第一行                   |
| `G`    | 跳转到文件结尾   | 跳转到文件的最后一行                 |
| `{n}G` | 跳转到第 n 行    | 跳转到指定的行号                     |
| `:{n}` | 跳转到第 n 行    | 使用命令行跳转到指定行号             |
| `%`    | 跳转到匹配的括号 | 在配对的括号、花括号、方括号之间跳转 |
| `*`    | 向前查找当前单词 | 向前搜索光标下的单词                 |
| `#`    | 向后查找当前单词 | 向后搜索光标下的单词                 |
| `gd`   | 跳转到局部定义   | 跳转到当前变量的定义位置             |
| `gD`   | 跳转到全局定义   | 跳转到全局变量的定义位置             |

### 数字前缀

Vim 支持在移动命令前加数字来重复执行操作：

- `3w`：向前移动 3 个单词
- `5j`：向下移动 5 行
- `10l`：向右移动 10 个字符

这种数字前缀可以大幅提升移动效率，特别是在需要精确移动时。

## 第三部分：文本编辑

文本编辑是 Vim 的核心功能之一。Vim 提供了丰富的编辑命令，通过操作符和动作的组合，可以实现高效的文本操作。

### 删除操作

| 快捷键      | 功能               | 说明                                 |
| ----------- | ------------------ | ------------------------------------ |
| `x`         | 删除当前字符       | 删除光标下的字符                     |
| `X`         | 删除前一个字符     | 删除光标前的字符                     |
| `d{motion}` | 删除操作           | 删除指定范围的文本（操作符）         |
| `dd`        | 删除当前行         | 删除整行                             |
| `D`         | 删除到行尾         | 删除从光标位置到行尾的内容           |
| `dw`        | 删除到单词结尾     | 删除从光标到下一个单词开头的文本     |
| `de`        | 删除到单词结尾     | 删除从光标到当前单词结尾的文本       |
| `d$`        | 删除到行尾         | 删除从光标到行尾的文本               |
| `d0`        | 删除到行首         | 删除从光标到行首的文本               |
| `d^`        | 删除到行首非空字符 | 删除从光标到行首第一个非空字符的文本 |

**删除操作说明**：`d` 是删除操作符，可以与其他动作组合使用。删除的内容会被保存到默认寄存器中，可以使用 `p` 粘贴。

### 复制操作

| 快捷键      | 功能               | 说明                                 |
| ----------- | ------------------ | ------------------------------------ |
| `y{motion}` | 复制操作           | 复制指定范围的文本（操作符）         |
| `yy`        | 复制当前行         | 复制整行                             |
| `Y`         | 复制当前行         | 等同于 `yy`                          |
| `yw`        | 复制到单词结尾     | 复制从光标到下一个单词开头的文本     |
| `ye`        | 复制到单词结尾     | 复制从光标到当前单词结尾的文本       |
| `y$`        | 复制到行尾         | 复制从光标到行尾的文本               |
| `y0`        | 复制到行首         | 复制从光标到行首的文本               |
| `y^`        | 复制到行首非空字符 | 复制从光标到行首第一个非空字符的文本 |

**复制操作说明**：`y`（yank）是复制操作符，复制的内容会被保存到默认寄存器中。

### 粘贴操作

| 快捷键 | 功能           | 说明                           |
| ------ | -------------- | ------------------------------ |
| `p`    | 在光标后粘贴   | 在光标位置后粘贴内容           |
| `P`    | 在光标前粘贴   | 在光标位置前粘贴内容           |
| `gp`   | 粘贴并移动光标 | 粘贴后光标移动到粘贴内容的末尾 |
| `gP`   | 粘贴并移动光标 | 粘贴后光标移动到粘贴内容的开头 |

**粘贴技巧**：

- 粘贴整行时，`p` 会在当前行下方粘贴，`P` 会在当前行上方粘贴
- 粘贴单词或字符时，`p` 会在光标后粘贴，`P` 会在光标前粘贴

### 撤销与重做

| 快捷键   | 功能           | 说明                   |
| -------- | -------------- | ---------------------- |
| `u`      | 撤销上一步操作 | 撤销最后一次编辑操作   |
| `Ctrl-r` | 重做撤销的操作 | 恢复被撤销的操作       |
| `U`      | 撤销整行的修改 | 撤销对当前行的所有修改 |

**撤销树**：Vim 支持撤销树功能，可以撤销到历史编辑状态。使用 `:undolist` 可以查看撤销历史。

### 替换操作

| 快捷键      | 功能                         | 说明                               |
| ----------- | ---------------------------- | ---------------------------------- |
| `r{char}`   | 替换单个字符                 | 用指定字符替换光标下的字符         |
| `R`         | 进入替换模式                 | 进入替换模式，逐个替换字符         |
| `s`         | 删除字符并进入插入模式       | 删除光标下的字符并进入插入模式     |
| `S`         | 删除整行并进入插入模式       | 删除当前行并进入插入模式           |
| `cc`        | 删除整行并进入插入模式       | 等同于 `S`                         |
| `c{motion}` | 删除并进入插入模式           | 删除指定范围并进入插入模式         |
| `cw`        | 删除到单词结尾并进入插入模式 | 删除从光标到单词结尾并进入插入模式 |
| `ce`        | 删除到单词结尾并进入插入模式 | 等同于 `cw`                        |
| `c$`        | 删除到行尾并进入插入模式     | 删除从光标到行尾并进入插入模式     |

**替换模式 vs 插入模式**：`R` 进入替换模式，会直接覆盖原有字符；`i` 进入插入模式，会在字符前插入。

### 大小写转换

| 快捷键       | 功能           | 说明                     |
| ------------ | -------------- | ------------------------ |
| `~`          | 切换字符大小写 | 切换光标下字符的大小写   |
| `g~{motion}` | 切换范围大小写 | 切换指定范围文本的大小写 |
| `gu{motion}` | 转换为小写     | 将指定范围文本转换为小写 |
| `gU{motion}` | 转换为大写     | 将指定范围文本转换为大写 |
| `guu`        | 当前行转小写   | 将当前行转换为小写       |
| `gUU`        | 当前行转大写   | 将当前行转换为大写       |

## 第四部分：搜索与替换

搜索和替换是文本编辑中的常用操作。Vim 提供了强大的搜索和替换功能，支持正则表达式，可以高效地查找和修改文本。

### 搜索操作

| 快捷键     | 功能             | 说明                       |
| ---------- | ---------------- | -------------------------- |
| `/pattern` | 向前搜索         | 从光标位置向前搜索指定模式 |
| `?pattern` | 向后搜索         | 从光标位置向后搜索指定模式 |
| `n`        | 下一个匹配       | 跳转到下一个搜索结果       |
| `N`        | 上一个匹配       | 跳转到上一个搜索结果       |
| `*`        | 向前搜索当前单词 | 向前搜索光标下的单词       |
| `#`        | 向后搜索当前单词 | 向后搜索光标下的单词       |
| `g*`       | 向前搜索部分匹配 | 向前搜索包含当前单词的文本 |
| `g#`       | 向后搜索部分匹配 | 向后搜索包含当前单词的文本 |

**搜索技巧**：

- 使用 `:set hlsearch` 可以高亮显示搜索结果
- 使用 `:set incsearch` 可以增量搜索，输入时实时显示匹配结果
- 使用 `:nohlsearch` 或 `:noh` 可以临时取消高亮
- 搜索历史可以使用上下箭头键浏览

### 替换操作

| 命令                  | 功能                 | 说明                           |
| --------------------- | -------------------- | ------------------------------ |
| `:s/old/new/`         | 替换当前行第一个匹配 | 在当前行替换第一个匹配的文本   |
| `:s/old/new/g`        | 替换当前行所有匹配   | 在当前行替换所有匹配的文本     |
| `:s/old/new/gc`       | 替换并确认           | 替换时逐个确认                 |
| `:%s/old/new/g`       | 替换全文所有匹配     | 在整个文件中替换所有匹配的文本 |
| `:%s/old/new/gc`      | 替换全文并确认       | 在整个文件中替换并逐个确认     |
| `:{range}s/old/new/g` | 替换指定范围         | 在指定行范围内替换             |

**替换范围**：

- `:1,10s/old/new/g`：替换第 1 到 10 行
- `:.,$s/old/new/g`：替换从当前行到文件末尾
- `:'<,'>s/old/new/g`：替换选中的行（可视模式）

### 正则表达式基础

Vim 支持强大的正则表达式，常用的模式包括：

| 模式    | 说明                       | 示例                           |
| ------- | -------------------------- | ------------------------------ |
| `.`     | 匹配任意字符               | `a.c` 匹配 "abc"、"a1c" 等     |
| `*`     | 匹配前一个字符 0 次或多次  | `ab*` 匹配 "a"、"ab"、"abb" 等 |
| `\+`    | 匹配前一个字符 1 次或多次  | `ab\+` 匹配 "ab"、"abb" 等     |
| `\?`    | 匹配前一个字符 0 次或 1 次 | `ab\?` 匹配 "a"、"ab"          |
| `^`     | 匹配行首                   | `^hello` 匹配行首的 "hello"    |
| `$`     | 匹配行尾                   | `world$` 匹配行尾的 "world"    |
| `\<`    | 匹配单词开头               | `\<the` 匹配单词开头的 "the"   |
| `\>`    | 匹配单词结尾               | `the\>` 匹配单词结尾的 "the"   |
| `[abc]` | 匹配字符类                 | `[abc]` 匹配 "a"、"b" 或 "c"   |
| `[0-9]` | 匹配数字范围               | `[0-9]` 匹配任意数字           |
| `\d`    | 匹配数字                   | 等同于 `[0-9]`                 |
| `\w`    | 匹配单词字符               | 匹配字母、数字和下划线         |
| `\s`    | 匹配空白字符               | 匹配空格、制表符等             |

**替换中的特殊字符**：

- `&`：代表匹配的整个文本
- `\1`、`\2` 等：代表捕获组
- `\u`：将下一个字符转换为大写
- `\U`：将后续字符转换为大写，直到 `\E`
- `\l`：将下一个字符转换为小写
- `\L`：将后续字符转换为小写，直到 `\E`

**示例**：

```vim
" 将 "hello" 替换为 "Hello"
:%s/hello/Hello/g

" 交换两个单词的位置（使用捕获组）
:%s/\(word1\) \(word2\)/\2 \1/g

" 在每行开头添加 "// "
:%s/^/\/\/ /

" 删除行尾的空格
:%s/\s\+$//g
```

## 第五部分：文件与缓冲区操作

Vim 支持同时编辑多个文件，通过缓冲区和窗口管理功能，可以高效地在多个文件间切换和编辑。

| `:wq`         | 保存并退出   | 保存文件后退出                       |

| `:e filename` | 打开文件     | 在当前窗口打开文件                   |
| `:e!`         | 重新加载文件 | 放弃修改，重新加载文件               |
| `:w!`         | 强制保存     | 强制保存文件（只读文件需要）         |

| 命令          | 功能                 | 说明                                 |

## 第七部分：高级技巧

Vim 的高级功能包括操作符、文本对象、寄存器、宏录制等，这些功能可以大幅提升编辑效率，实现复杂的编辑操作。

### 操作符

操作符是 Vim 中用于对文本执行操作的命令。操作符通常需要配合动作（motion）或文本对象使用。

| 操作符 | 功能       | 说明                       |
| ------ | ---------- | -------------------------- |
| `d`    | 删除       | 删除指定范围的文本         |
| `c`    | 修改       | 删除指定范围并进入插入模式 |
| `y`    | 复制       | 复制指定范围的文本         |
| `=`    | 格式化     | 格式化指定范围的代码       |
| `>`    | 右缩进     | 增加指定范围的缩进         |
| `<`    | 左缩进     | 减少指定范围的缩进         |
| `g~`   | 大小写转换 | 切换指定范围的大小写       |
| `gu`   | 转小写     | 将指定范围转换为小写       |
| `gU`   | 转大写     | 将指定范围转换为大写       |

**操作符使用模式**：`{operator}{motion}` 或 `{operator}{text-object}`

**示例**：

- `dw`：删除到下一个单词开头
- `ci"`：删除引号内的内容并进入插入模式
- `>ap`：增加段落的缩进
- `gUiw`：将当前单词转换为大写

### 文本对象

文本对象是 Vim 中用于选择特定文本范围的机制。文本对象通常以 `i`（inner，内部）或 `a`（around，包含边界）开头。

| 文本对象     | 功能                 | 说明                         |
| ------------ | -------------------- | ---------------------------- |
| `iw`         | 内部单词             | 选择单词（不含空格）         |
| `aw`         | 包含单词             | 选择单词（含前后空格）       |
| `iW`         | 内部单词（空格分隔） | 选择空格分隔的单词           |
| `aW`         | 包含单词（空格分隔） | 选择空格分隔的单词（含空格） |
| `is`         | 内部句子             | 选择句子                     |
| `as`         | 包含句子             | 选择句子（含标点）           |
| `ip`         | 内部段落             | 选择段落                     |
| `ap`         | 包含段落             | 选择段落（含空行）           |
| `i"`         | 内部引号             | 选择双引号内的内容           |
| `a"`         | 包含引号             | 选择双引号及其内容           |
| `i'`         | 内部单引号           | 选择单引号内的内容           |
| `a'`         | 包含单引号           | 选择单引号及其内容           |
| `i)` 或 `ib` | 内部括号             | 选择圆括号内的内容           |
| `a)` 或 `ab` | 包含括号             | 选择圆括号及其内容           |
| `i]`         | 内部方括号           | 选择方括号内的内容           |
| `a]`         | 包含方括号           | 选择方括号及其内容           |
| `i}` 或 `iB` | 内部花括号           | 选择花括号内的内容           |
| `a}` 或 `aB` | 包含花括号           | 选择花括号及其内容           |
| `it`         | 内部标签             | 选择 HTML/XML 标签内的内容   |
| `at`         | 包含标签             | 选择 HTML/XML 标签及其内容   |

**文本对象使用示例**：

- `di"`：删除双引号内的内容
- `ci(`：删除圆括号内的内容并进入插入模式
- `yi{`：复制花括号内的内容
- `>ap`：增加段落的缩进
- `vat`：选择整个 HTML 标签

### 寄存器

寄存器是 Vim 中用于存储文本的存储区域。Vim 提供了多种类型的寄存器，用于不同的用途。

| 寄存器类型             | 说明                 | 使用方式                                            |
| ---------------------- | -------------------- | --------------------------------------------------- |
| 无名寄存器 `"`         | 默认寄存器           | 自动使用，无需指定                                  |
| 数字寄存器 `"0` - `"9` | 存储删除和复制的内容 | `"0` 存储最近复制的内容，`"1` - `"9` 存储删除的内容 |
| 命名寄存器 `"a` - `"z` | 用户自定义寄存器     | 可以存储任意内容                                    |
| 只读寄存器 `".`        | 最后插入的文本       | 只读，存储最后插入的文本                            |
| 只读寄存器 `":`        | 最后执行的命令       | 只读，存储最后执行的命令                            |
| 只读寄存器 `"%`        | 当前文件名           | 只读，存储当前文件名                                |
| 只读寄存器 `"#`        | 轮换文件名           | 只读，存储轮换文件名                                |
| 表达式寄存器 `"=`      | 表达式求值           | 可以执行表达式并插入结果                            |
| 选择寄存器 `"*`        | 系统剪贴板（X11）    | 系统剪贴板（Linux）                                 |
| 选择寄存器 `"+`        | 系统剪贴板           | 系统剪贴板（Windows/macOS）                         |
| 黑洞寄存器 `"_`        | 删除但不保存         | 删除的内容不会保存                                  |

**寄存器使用示例**：

```vim
" 复制到命名寄存器
"ayy          " 将当前行复制到寄存器 a
"ap           " 粘贴寄存器 a 的内容

" 删除到命名寄存器
"bdw          " 删除单词到寄存器 b

" 追加到寄存器
"Ayy          " 追加当前行到寄存器 a

" 查看寄存器内容
:reg          " 查看所有寄存器
:reg a        " 查看寄存器 a
```

### 宏录制

宏录制是 Vim 中用于记录和重复一系列操作的功能。通过宏录制，可以将复杂的编辑操作记录下来，然后在多个位置重复执行。

| 操作             | 功能        | 说明                   |
| ---------------- | ----------- | ---------------------- |
| `q{register}`    | 开始录制宏  | 开始录制宏到指定寄存器 |
| `q`              | 停止录制    | 停止录制宏             |
| `@{register}`    | 执行宏      | 执行指定寄存器的宏     |
| `@@`             | 重复上次宏  | 重复执行上一次执行的宏 |
| `{n}@{register}` | 执行宏 n 次 | 重复执行宏 n 次        |

**宏录制示例**：

```vim
" 录制宏：在每行开头添加 "// "
qa            " 开始录制宏到寄存器 a
I// <Esc>     " 在行首插入 "// " 并返回普通模式
j             " 移动到下一行
q             " 停止录制

" 执行宏
@a            " 执行寄存器 a 的宏
5@a           " 执行 5 次宏
```

**宏使用技巧**：

- 在录制宏时，确保操作可以重复执行
- 使用数字前缀可以重复执行宏
- 宏可以嵌套使用（在宏中执行另一个宏）
- 使用 `:reg` 可以查看宏的内容

### 标记

标记是 Vim 中用于标记文件中特定位置的机制。通过标记，可以快速跳转到之前编辑的位置。

| 命令           | 功能               | 说明                         |
| -------------- | ------------------ | ---------------------------- |
| `m{letter}`    | 设置标记           | 在当前光标位置设置标记       |
| `'{letter}`    | 跳转到标记行首     | 跳转到标记所在行的第一个字符 |
| `` `{letter}`` | 跳转到标记位置     | 跳转到标记的精确位置         |
| `` ` ``        | 跳转到上次位置     | 跳转到上次编辑的位置         |
| `` `. ``       | 跳转到上次修改位置 | 跳转到上次修改的位置         |
| `` `[ ``       | 跳转到上次修改开始 | 跳转到上次修改的开始位置     |
| `` `] ``       | 跳转到上次修改结束 | 跳转到上次修改的结束位置     |
| `:marks`       | 列出所有标记       | 显示所有标记的位置           |

**标记类型**：

- 小写字母标记（`a` - `z`）：文件内标记，仅在当前文件有效
- 大写字母标记（`A` - `Z`）：文件间标记，跨文件有效
- 数字标记（`0` - `9`）：由 Vim 自动设置，记录最近访问的位置
- 特殊标记：如 `` `. ``、`` `[ ``、`` `] `` 等

**标记使用示例**：

```vim
ma            " 在当前位置设置标记 a
'a            " 跳转到标记 a 所在行的行首
`a            " 跳转到标记 a 的精确位置
d'a           " 删除从当前位置到标记 a 的内容
y`a           " 复制从当前位置到标记 a 的内容
```

### 代码折叠

代码折叠是 Vim 中用于隐藏和显示代码块的功能，特别适合编辑大型文件。

## 结语

Vim 的快捷键系统是一个强大而灵活的工具，掌握这些快捷键可以显著提升文本编辑效率。本文档涵盖了 Vim 的主要快捷键和功能，从基础的光标移动到高级的宏录制，为不同水平的用户提供了全面的参考。

### 学习路径建议

**初学者**：

1. 掌握基础光标移动（`h`、`j`、`k`、`l`、`w`、`e`、`b`）

3. 熟悉模式切换（`Esc`、`i`、`v`、`:`）
4. 掌握文件操作（`:w`、`:q`、`:wq`）

**中级用户**：

1. 深入学习操作符和文本对象（`d`、`c`、`y`、`iw`、`ap`）
2. 掌握搜索和替换（`/`、`?`、`:s`、`:%s`）
3. 学习窗口和标签页管理
4. 熟悉寄存器使用

**高级用户**：

### 实践建议

- **每日练习**：每天花 10-15 分钟练习 Vim 快捷键，形成肌肉记忆
- **实际应用**：在日常编辑任务中使用 Vim，逐步替代传统编辑器
- **循序渐进**：不要试图一次性掌握所有快捷键，逐步学习并巩固
- **参考文档**：遇到问题时查阅 Vim 帮助文档（`:help`）

- **Vim 官方文档**：`:help` 命令查看内置帮助
