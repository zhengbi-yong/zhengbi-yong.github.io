## 项目概述

**Newton** 是一个开源的 GPU 加速物理模拟引擎，专为机器人学家和模拟研究人员设计。该项目基于 **NVIDIA Warp** 构建，是对 Warp 已弃用的 `warp.sim` 模块的扩展和泛化，并集成了 **MuJoCo Warp** 作为主要后端。

### 基本信息

| 属性 | 详情 |
|------|------|
| **发起机构** | Disney Research, Google DeepMind, NVIDIA |
| **管理组织** | Linux Foundation |
| **许可证** | Apache-2.0 |
| **开发状态** | Beta（活跃开发，API 不稳定） |
| **Python 版本** | 3.10, 3.11, 3.12 |
| **平台** | Linux, Windows (x86-64) |
| **GPU 要求** | NVIDIA CUDA (Compute Capability ≥ 5.0, Maxwell 架构) |

### 技术基础

Newton 建立在两个核心技术之上：

1. **NVIDIA Warp** - 用于编写高性能模拟和计算代码的 Python 框架
2. **MuJoCo-Warp** - Google DeepMind 开发的 GPU 优化版 MuJoCo
3. **OpenUSD** - Pixar 开发的通用场景描述标准

## 核心特性

### 1. GPU 加速仿真

Newton 的核心优势是其出色的 GPU 加速能力：

- **大规模并行**：可同时仿真数千个环境实例
- **高性能计算**：优化的 GPU 代码充分利用 GPU 计算能力
- **可扩展性**：性能随 GPU 核心数量线性扩展

### 2. 可微分物理

支持梯度计算和基于梯度的优化：

- **策略梯度学习**：可直接计算策略梯度用于强化学习
- **参数估计**：从观测数据中估计物理参数
- **轨迹优化**：优化机器人的运动轨迹
- **逆动力学**：计算达到目标状态所需的控制输入

### 3. 模块化架构

```python
from newton import ModelBuilder, Model, State

# 构建模型
builder = ModelBuilder()
builder.add_articulation(...)
model = builder.finalize()

# 创建状态
state = model.state()
```

**设计特点:**
- **Model**: 存储所有静态数据（几何体、约束、参数）
- **State**: 管理动态时变数据（位置、速度）
- **Control**: 处理控制输入

### 4. 多物理求解器

| 求解器 | 用途 |
|--------|------|
| `SolverMuJoCo` | 刚体动力学（基于 MuJoCo） |
| `SolverFeatherstone` | 刚体动力学（Featherstone 算法） |
| `SolverSemiImplicit` | 半隐式欧拉积分 |
| `SolverXPBD` | 位置动力学（XPBD） |
| `SolverVBD` | 速度动力学（VBD） |
| `SolverImplicitMPM` | 隐式 MPM 求解器 |
| `SolverStyle3D` | Style3D 服装模拟 |

### 5. 传感器系统

```python
from newton.sensors import ContactSensor, RaycastSensor

# 接触传感器
contact_sensor = ContactSensor(model, shape_indices=[...])
contact = contact_sensor.eval(model, state)

# 光线投射传感器
raycast_sensor = RaycastSensor(model, origins, directions)
depth, _ = raycast_sensor.eval(model, state)
```

### 6. 几何和碰撞检测

- **SDF** (Signed Distance Fields): 有符号距离场
- **Mesh**: 网格几何体
- **GJK 算法**: Gilbert-Johnson-Keerthi 碰撞检测
- **Broad Phase**: 空间分割和包围体层次
- **Raycast**: 光线投射查询

## Model 和 State 架构

### Model 类

`Model` 类封装了模拟系统的所有静态（非时变）定义：

```python
class Model:
    """模拟模型的静态定义"""

    # 粒子系统
    particle_q: wp.array          # 粒子位置
    particle_qd: wp.array         # 粒子速度
    particle_mass: wp.array       # 粒子质量

    # 刚体系统
    body_q: wp.array              # 刚体位姿
    body_qd: wp.array             # 刚体速度
    body_mass: wp.array           # 刚体质量
    body_inertia: wp.array        # 刚体惯性张量

    # 关节系统
    joint_q: wp.array             # 关节位置
    joint_qd: wp.array            # 关节速度
    joint_type: wp.array          # 关节类型

    # 几何形状
    shape_transform: wp.array     # 形状变换
    shape_type: wp.array          # 形状类型
    shape_material_mu: wp.array   # 摩擦系数

    # 世界索引
    particle_world: wp.array      # 粒子所属世界
    body_world: wp.array          # 刚体所属世界
```

**关键特性:**
- **静态数据**: 存储几何体、约束、参数
- **世界分组**: 支持多世界并行模拟
- **设备管理**: 所有数据在同一设备上

### State 类

```python
state = model.state(requires_grad=True)

# State 包含:
# - particle_q, particle_qd (粒子状态)
# - body_q, body_qd (刚体状态)
# - joint_q, joint_qd (关节状态)
```

### Control 类

```python
control = model.control(requires_grad=True)

# Control 包含:
# - joint_target (关节目标)
# - joint_f (关节力)
# - tri_activations (三角形激活)
# - tet_activations (四面体激活)
# - muscle_activations (肌肉激活)
```

### 模拟循环

```python
from newton import ModelBuilder, State
from newton.solvers import SolverMuJoCo

# 构建模型
builder = ModelBuilder()
# ... 添加物体、关节等
model = builder.finalize()

# 创建状态
state = model.state()
control = model.control()

# 模拟循环
solver = SolverMuJoCo(model)
for step in range(num_steps):
    # 更新控制输入
    control.joint_target[...] = target_values

    # 推进模拟
    state = solver.step(model, state, control)
```

## 技术亮点

### 1. 多世界并行模拟

Newton 支持在单个 Model 中创建多个独立的世界：

```python
# 世界索引: -1 表示全局, 0, 1, 2... 表示特定世界
model.particle_world  # 每个粒子的世界索引
model.body_world      # 每个刚体的世界索引

# 优势:
# - 碰撞检测优化（分离世界）
# - 可视化（空间分离）
# - 并行处理（独立世界）
```

### 2. 可微物理

```python
# 启用梯度计算
model = builder.finalize(requires_grad=True)
state = model.state(requires_grad=True)

# 计算梯度
loss = compute_loss(state)
loss.backward()

# 应用: 策略学习、参数估计、轨迹优化
```

### 3. 资产导入

支持多种模型格式：

| 格式 | 描述 | 用途 |
|------|------|------|
| **URDF** | 统一机器人描述格式 | ROS 生态系统 |
| **MJCF** | MuJoCo XML 格式 | MuJoCo 模型 |
| **USD** | Universal Scene Description | 复杂场景描述 |

```python
from newton.utils import import_urdf, import_mjcf, import_usd

# URDF 导入
model, builder = import_urdf("robot.urdf")

# MJCF 导入
model, builder = import_mjcf("robot.xml")

# USD 导入
model, builder = import_usd("scene.usd")
```

### 4. 可视化选项

| 查看器 | 描述 | 用途 |
|--------|------|------|
| `gl` | OpenGL 实时查看器 | 交互式可视化 |
| `usd` | USD 文件输出 | 离线渲染、场景编辑 |
| `rerun` | ReRun 集成 | 数据可视化和调试 |
| `null` | 无头模式 | 纯计算、无可视化 |

```python
from newton.viewer import ViewerGL, ViewerUSD, ViewerRerun, ViewerNull

# OpenGL 查看器
viewer = ViewerGL()

# USD 查看器
viewer = ViewerUSD(output_path="output.usd")

# ReRun 查看器
viewer = ViewerRerun()

# 无头模式
viewer = ViewerNull()
```

## 总结

Newton 是一个功能强大、设计现代化的物理模拟引擎，具有以下优势：

### 核心优势

1. **高性能** - GPU 加速实现大规模并行模拟
2. **灵活性** - 多种求解器支持不同物理场景
3. **可扩展** - 用户可自定义组件和求解器
4. **易用性** - 清晰的 API 设计和丰富的示例
5. **开源** - Apache-2.0 许可，社区驱动开发

### 与其他引擎对比

| 特性 | Newton | MuJoCo | Isaac Sim | Brax |
|------|--------|--------|-----------|------|
| **GPU 加速** | ✅ | ❌ (基础版) | ✅ | ✅ |
| **可微物理** | ✅ | ❌ | 部分 | ✅ |
| **多求解器** | ✅ | ❌ | ❌ | ❌ |
| **OpenUSD** | ✅ | ❌ | ✅ | ❌ |
| **开源** | ✅ | ✅ | ❌ | ✅ |

### 发展前景

Newton 目前处于 **Beta 阶段**，API 可能会有破坏性变更，但已经展示出在机器人仿真领域的巨大潜力。对于需要进行大规模、高保真物理模拟的研究人员和工程师来说，Newton 是一个值得关注和投入的项目。

随着 GPU 计算能力的不断提升和物理仿真技术的持续发展，Newton 这样的 GPU 加速、可微分仿真引擎将在机器人学和强化学习研究中发挥越来越重要的作用。
