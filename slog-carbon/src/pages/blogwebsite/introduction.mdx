## 快速开始
* 从[GitHub](https://github.com/zhengbi-yong/zhengbi-yong.github.io.git)仓库中将网站的文件克隆到本地
```powershell
git clone zhengbi-yong/zhengbi-yong.github.io.git
```
* 在本地配置好npm和yarn
* 删除node_modules文件夹和yarn.lock文件
* 运行`gatsby develop`
