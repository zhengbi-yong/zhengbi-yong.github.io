# 数据隐私与合规（规划）

> ⚠️ **重要提示**: 本文档描述的所有功能和 API 均 **尚未实施**。以下内容仅为未来规划。当前代码库中不存在 cookie 同意、用户数据导出、数据删除或自动保留清理功能。

> 来源：EDITOR_SYSTEM_DESIGN.md P8 — 扩展阶段，尚未实施

## 合规要求

| 法规 | 适用范围 | 核心要求 |
|------|---------|----------|
| GDPR | EU 用户 | 数据可删除、可导出、明确同意 — ❌ cookie 同意横幅未实施 |
| 个人信息保护法 | 中国用户 | 最小必要、告知同意、跨境传输 |
| CCPA | 加州用户 | 知情权、删除权、不出售权 |

## 数据分类

| 类别 | 示例 | 处理方式 |
|------|------|---------|
| 个人身份信息 | 邮箱、IP 地址 | 加密存储、可清除 |
| 认证数据 | 密码哈希、Token | Argon2id 哈希、不可逆 |
| 内容数据 | 文章、评论 | 保留直到用户删除 |
| 行为数据 | 浏览历史、搜索记录 | 匿名化、保留有限期 |
