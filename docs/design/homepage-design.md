# 首页设计

> ⚠️ **未实现 — 设计规划**：本文档描述的沉浸式体验方案（WebGL 粒子、Bento Grid、Projects Gallery、Music Experience、Mega Footer、自定义鼠标指针）属于早期设计规划，**尚未实现**。当前实际首页使用 "visitor theme" 布局。下文以「当前实现」标注实际存在的组件。

## 当前首页布局

实际首页由以下简单组件构成，采用访客主题（Visitor Theme）布局：

```
┌───────────────────────────────────────────┐
│               HeroCard                     │
│          (头像 + 简介 + CTA)               │
├───────────────────────────────────────────┤
│              SocialCard                    │
│          (社交链接展示区域)                 │
├───────────────────────────────────────────┤
│               Explore                      │
│          (探索导航区域)                     │
├───────────────────────────────────────────┤
│            FeaturedWork                   │
│          (精选项目展示)                     │
├───────────────────────────────────────────┤
│            BlogSection                     │
│          (最新博客文章列表)                 │
├───────────────────────────────────────────┤
│          NewsletterSignup                 │
│          (邮件订阅)                        │
└───────────────────────────────────────────┘
```

## 组件说明

### HeroCard
- 头像 + 姓名/标题 + 简短简介
- CTA 按钮引导至主要页面

### SocialCard
- 展示社交链接（GitHub、Twitter 等）
- 图标 + 链接

### Explore
- 导航区域，链接到博客、项目、关于等页面

### FeaturedWork
- 精选项目卡片展示
- 简单的网格或列表布局

### BlogSection
- 最新文章列表
- 标题 + 摘要 + 日期

### NewsletterSignup
- 邮件订阅表单
- 输入框 + 提交按钮

## 设计规划（未实现）

以下为早期规划但未实现的设计概念，仅供存档参考：

### 沉浸式 Hero（未实现）
- **背景**：Three.js 流体粒子云（桌面 2000-3000，移动端 500）
- 粒子模拟有机波/数据流运动，颜色在靛蓝/紫/琥珀间渐变的
- 主标题：`clamp(3rem, 8vw, 8rem)`，衬线字体
- 入场动画：GSAP SplitText，逐字淡入+模糊→清晰

### Bento Grid（未实现）
- CSS Grid: 桌面 4 列、平板 2 列、移动端 1 列
- Glassmorphism 2.0 视觉风格
- 跨距网格展示最新内容

### Projects Gallery（未实现）
- 横向滚动画廊：CSS `scroll-snap-type: x mandatory`
- 卡片 4:5 宽高比
- Framer Motion 拖拽 + 3D 倾斜 Hover

### Music Experience（未实现）
- 左右分屏：左 60% 矢量乐谱 + 右 40% 曲目控制
- Three.js 音频响应粒子系统
- Web Audio API 实时频谱

### Mega Footer（未实现）
- 全屏 `100vh`，深色背景 `#050505`
- 巨幅声明文字

### 自定义鼠标指针（未实现）
- 默认小点 + 外圈
- 不同元素显示不同图标
