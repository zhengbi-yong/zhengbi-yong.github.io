## ⚠️ Requirements / 要求

- **RAM / 内存**: 2GB (exact / 精确)
- **Disk / 磁盘**: 20GB SSD (SSD required / 必须SSD)
- **CPU / 处理器**: 1-2 cores
- **Swap / 交换空间**: 2GB recommended / 推荐

## 🎯 Is Low Resource Right for You? / 低配置适合您吗?

**Yes / 是** if / 如果:
- Budget is tight / 预算紧张 ($3-5/month)
- Traffic is low / 流量较低 (<1,000 daily visitors / <1,000日访客)
- Learning deployment / 学习部署

**No / 否** if / 如果:
- You can afford $5-10/month / 您负担得起$5-10/月
- Need better performance / 需要更好的性能
- Plan to grow quickly / 计划快速增长

→ Consider [Single Server](../server/single-server.md) instead / 改为考虑[单服务器](../server/single-server.md)
