# MDX Migration Script (Archived)

## Module Overview

Legacy migration script for transitioning MDX content between crate systems.

## Status

**ARCHIVED** - This script is preserved for reference but no longer actively maintained.

## Purpose

Historical utility for migrating MDX (Markdown + JSX) content from old crate structure to new system.

## Components

- MDX parsing
- Content transformation
- Metadata extraction
- File system operations

## Migration Path

Content now handled by:
- `/backend/crates/core` - Core content processing
- `/backend/crates/api` - API endpoints
- `/frontend/src/lib/mdx` - Frontend MDX rendering

## Historical Context

Used during major refactoring to consolidate content management systems.
