# Excalidraw 功能使用指南

## 功能修复说明

已修复 Excalidraw 内置菜单的"保存"和"导出"功能，现在它们可以正常使用了。

## 主要功能

### 1. 保存功能

- **内置保存按钮**：点击内置菜单的保存按钮即可自动保存
- **自动保存位置**：绘图数据会自动保存到浏览器的 localStorage
- **保存成功提示**：右上角会显示绿色的成功提示

### 2. 导出功能

支持三种导出格式：

- **导出为 PNG**：导出为高质量的图片文件
- **导出为 SVG**：导出为矢量图形，可无损缩放
- **导出为 JSON**：导出完整的绘图数据，可用于后续编辑

### 3. 键盘快捷键

为了提高使用效率，提供了以下快捷键：

- **Ctrl/Cmd + S**：保存绘图
- **Ctrl/Cmd + Shift + E**：导出为 PNG
- **Ctrl/Cmd + Shift + S**：导出为 SVG
- **Ctrl/Cmd + Shift + J**：导出为 JSON

### 4. 自动恢复

- 打开页面时会自动加载上次保存的绘图
- 如果有未保存的更改，系统会提示是否保存

## 使用技巧

1. **定期保存**：虽然支持自动保存，但建议使用快捷键 Ctrl+S 定期保存
2. **导出选择**：
   - 需要图片分享时使用 PNG 导出
   - 需要进一步编辑或缩放时使用 SVG 导出
   - 需要备份或转移时使用 JSON 导出
3. **主题适配**：导出的文件会自动适配当前的主题（亮色/暗色）

## 注意事项

- 导出功能依赖于现代浏览器 API，建议使用最新版本的浏览器
- localStorage 有限制，大量绘图可能会覆盖旧的保存记录
- JSON 文件可以重新导入到 Excalidraw 中继续编辑

## 故障排除

如果遇到功能无法使用：

1. 检查浏览器控制台是否有错误信息
2. 尝试刷新页面重新加载
3. 确保浏览器支持 File API 和 Blob API
4. 检查浏览器是否阻止了文件下载
